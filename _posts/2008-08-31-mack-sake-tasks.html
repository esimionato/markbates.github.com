Now getting up and running with Mack development is even easier. Thanks DeLynn! All you need to do is install Sake. Once Sake is installed you can install the Mack sake tasks like so: 
sake -i http://www.mackframework.com/mack-dev.sake
You can see a full list of tasks with the following command: 
sake -T
